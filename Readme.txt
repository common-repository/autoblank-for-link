== Description ==

Плагин автоматически внедряет во все ссылки параметр target="_blank" в тексте статьи.

P.S: Если у вас остались какие-либо вопросы пишите на почту <cite>info[@]artemsannikov.ru</cite>.

== Installation ==
	
1. Загрузите плагин "Autoblank for link" в директорию `/wp-content/plugins/`;
2. Активируйте "Autoblank for link" на странице `Плагины` в панели управления.
